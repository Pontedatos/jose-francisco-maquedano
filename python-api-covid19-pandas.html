Comparacion¶

Compararemos estos tres pases en un solo grfico

Lo primero creamos una variable para cada uno de los dataframe de los tres pases que hemos escogido, en este caso Espaa, Italia y Alemania

casos_es = df_es.set_index("Date")["Cases"]
casos_it = df_it.set_index("Date")["Cases"]
casos_ger = df_ger.set_index("Date")["Cases"]
